# KDU Global University Backend API

Backend API for KDU Global University's faculty and admin management system.

## Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 👨‍🏫 Faculty Profile Management
- 📚 Publication Management
- 🏛️ Department Management
- 👑 Admin Dashboard

## Tech Stack

- Node.js
- Express.js
- MongoDB
- JWT Authentication
- bcryptjs for password hashing

## Installation

### Prerequisites
- Node.js (v16+)
- MongoDB
